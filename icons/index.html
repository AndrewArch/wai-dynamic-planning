Accessibility Planning Roadmap

This resource explores activities that are important when planning for accessibility. Activities are grouped by when in a project or program they might be considered. Each activity provides information on what is required and points to related resources that explore the activity further.

  • Initiate

    Develop clear goals and an environment supportive of accessibility.

    • Get Familiar
    • Make Baseline
    • Set Goals
    • Business Case
    • Get Support
    • Spread the Word

  • Plan

    Clarify scope based on an understanding of current situation.

    • Define Roles
    • Make a Policy
    • Assign Budget
    • Review Websites
    • Review Design Materials
    • Review Tools
    • Enhance Policies
    • Develop Milestones
    • Develop Reporting
    • Setup Feedback

  • Implement

    Ensure personal are trained, tools are available, and accessibility is included throughout.

    • Assign Roles
    • Train Staff
    • Recruit Experts
    • Setup Evaluation
    • Prioritize Issues
    • Share Knowledge
    • Procure Accessibility

  • Sustain

    Continue to review and report on content, processes, and resources.

    • Review Policies
    • Review Websites
    • Look To Improve
    • Get User Feedback
    • Report To Stakeholders
